Sandbox 自述¶

Sandbox 是一个可以体验、尝试和分享想法的地方。sandbox/ 目录是 Subversion 存储库的一部分，但它不是作为 Docutils 发行版的一部分分发的。

在 Sandbox 中搞得一团糟也没关系！但是，请友好一点。

应该为每个新项目创建一个特定于项目的子目录。任何想要使用沙箱的开发人员都可以创建他们自己的子目录（建议的名称：SourceForge ID，或者给定名称+family初始化），但是项目目录比个人目录更可取，这不利于协作。

有关更多信息，请参阅 Docutils Project Policies。沙盒镜像在 Docutils 网站上，因此 Docutils Web Site 文档同样适用
